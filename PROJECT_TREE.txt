🚀 QUICK START PATHS
═══════════════════════════════════════════════

To Play Game:
1. Open: project.godot in Godot Engine
2. Press: F5
3. Enjoy!

To Customize World Size:
→ scenes/main.tscn
  → Select: WorldGenerator node
  → Edit: World Size property

To Customize Player:
→ scripts/player.gd
  → Edit: speed, mouse_sensitivity

To Customize Camera:
→ scenes/player.tscn
  → Select: Camera3D node
  → Edit: Transform position
